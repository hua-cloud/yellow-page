## 项目介绍
`yellow-page`项目是一套基于`Spring Boot + Vue`开发的前后端分离的点评分享的平台，整体上类似于大众点评
主要功能包含基于短信验证码的登录，达人探店，优惠券秒杀，好友关注，用户签到以及附近的商铺信息等，本项目对
`Redis`的各种数据类型和相关技术进行了深度的应用，充分发挥了`Redis`的优势，不仅实现了相关功能同时提升系统整体性能。
